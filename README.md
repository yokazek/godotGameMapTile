## マップタイルのなかでジャンプ

マップタイルの実装の習作です。

- ルール
    - 画面をタップ、またはマウスクリックでその位置に応じたジャンプウを行います。手を離したタイミングでジャンプします。

以上です。 
個人的にキャラの動きがよくできた気がします。

あと画面左上でBGMのON/OFFができます。
